Installation

To install Frost Client, follow these steps:

  1. Download the Frost Client ZIP file from the official website.
  2. Extract the ZIP file.
  3. Open the Minecraft Launcher and select the Forge profile.
  4. Click on the "Mods" tab and open the mods folder.
  5. Copy the Frost Client JAR file into the mods folder.
  6. Launch Minecraft with the Forge profile, and Frost Client will be active.
